# S.C.A.U.T = Secret Creation And Uncovering Tool 

_A small program that will turn text in files into "secrets" that are only "uncoverable" with a key.
With it is attached a small example key. Make sure the key is longer than the content you wish to encrypt._

*DISCLAIMER: THIS ENCRYPTION IS WEAK (it is an OTP = One Time Pad), AND I DO NOT RECOMMEND USING IT TO ENCRYPT CONFIDENTIAL DATA!*
